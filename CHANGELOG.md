# Changelog

## [0.1.0] — 2026-04-30

### Added
- Initial design system component library with 40+ components
- Storybook documentation with interactive stories for all components
- Design token system (colors, typography, spacing, shadows)
- CI/CD pipeline with GitHub Actions (lint → test → build → deploy)
- Deploy workflow to GitHub Pages (docs site + Storybook)
- Contribution guidelines (CONTRIBUTING.md)
- Issue templates for bug reports and feature requests
- Accessibility annotations and guidelines

### Components
- Accordion, Alert, AutoRefreshIndicator, Avatar, Badge, Breadcrumb, Button
- Card, Checkbox, Chip, Collapsible, DataTable, DateTimePicker, DurationSlider
- EmptyState, ErrorBoundary, FileUpload, FilterBar, FormField, FormatSelect
- IconButton, Input, Modal, PageHeader, Pagination, ProgressBar, Radio
- Select, Skeleton, SlideOutPanel, SlidePanel, SocialIcon, Spinner
- Stepper, Tabs, Textarea, ThumbnailLightbox, Toast, ToggleSwitch, Tooltip
