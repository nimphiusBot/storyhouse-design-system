## 5 Bug Fixes: ORA-136, ORA-135, ORA-134, ORA-133, ORA-132

All 5 issues were already fixed in code (committed and pushed to `main`). Linear tickets transitioned to "In Review" with descriptive comments.

### ORA-136 — Modal focus trap
**Commit:** `94595c9`
**Fix:** Replaced custom inline focus-trap logic with shared `useFocusTrap` hook. Initial focus targets the primary action button (last non-close-button focusable) instead of the close button.

### ORA-135 — body scroll lock stacking
**Commit:** `f033c59`
**Fix:** Created shared `useBodyScrollLock` hook with module-level lock count. Replaced inline `document.body.style.overflow` in Modal, SlidePanel, and ThumbnailLightbox. Lock count ensures scroll is only restored when ALL stacked consumers release.

### ORA-134 — aria-sort on DataTable
**Commit:** `dd99337`
**Fix:** Added `aria-sort` attribute to `<th>` elements: `"ascending"`/`"descending"` for active sort column, `"none"` for other sortable columns, omitted for non-sortable.

### ORA-133 — DataTable sort state conflict
**Commit:** `e68fb3d`
**Fix:** Made DataTable sort state externally controlled when `onSort` is provided. `handleSort` now calls `onSort(key, direction)` without updating internal state. Eliminates conflicting controlled/uncontrolled pattern.

### ORA-132 — DurationSlider touch handlers
**Commit:** `5d960ee`
**Fix:** Added `onTouchStart`/`onTouchMove`/`onTouchEnd` handlers mirroring mouse drag. Uses `touch.clientY` for vertical sliders. Added `touch-action: none` CSS to prevent scroll interference.
